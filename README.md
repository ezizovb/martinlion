# Сайт интернет-магазина парфюмерии

Современный адаптивный сайт для продажи парфюмерии с минималистичным дизайном и удобной навигацией.

## Особенности

- ✅ Современный минималистичный дизайн
- ✅ Адаптивная верстка (мобильные, планшеты, десктопы)
- ✅ Поиск по товарам
- ✅ Фильтрация по категориям (Мужской, Женский, Унисекс)
- ✅ Фильтрация по брендам
- ✅ Автоматический слайдер баннеров
- ✅ Интеграция с EmailJS для отправки заказов
- ✅ Масштабируемая структура данных (JSON)

## Структура файлов

```
├── index.html      # Основная HTML структура
├── styles.css      # Стили и адаптивный дизайн
├── script.js       # JavaScript логика
├── data.json       # Данные товаров и брендов
└── README.md       # Документация
```

## Настройка EmailJS

Для работы функции заказа необходимо настроить EmailJS:

1. Зарегистрируйтесь на [EmailJS](https://www.emailjs.com/)
2. Создайте Email Service (Gmail, Outlook и т.д.)
3. Создайте Email Template с переменными:
   - `{{product_id}}`
   - `{{product_name}}`
   - `{{customer_name}}`
   - `{{customer_email}}`
   - `{{customer_phone}}`
   - `{{message}}`
4. Получите ваш Public Key, Service ID и Template ID
5. Откройте `script.js` и замените:
   - `YOUR_PUBLIC_KEY` на ваш Public Key (строка 2)
   - `YOUR_SERVICE_ID` на ваш Service ID (строка 150)
   - `YOUR_TEMPLATE_ID` на ваш Template ID (строка 151)

## Добавление товаров

Товары добавляются через файл `data.json`. Структура товара:

```json
{
  "id": "H19",
  "name": "Название парфюма",
  "brand": "Название бренда",
  "category": "male|female|unisex",
  "description": "Описание аромата",
  "image": "путь/к/изображению.jpg"
}
```

## Добавление изображений

1. Создайте папку `images/` в корне проекта
2. Добавьте изображения товаров
3. В `data.json` укажите путь к изображению в поле `image`:
   ```json
   "image": "images/product-name.jpg"
   ```

## Цветовая схема

- Основной цвет: белый (#ffffff)
- Дополнительный цвет: жемчужно-синий (#6B9BD1)
- Акцентный цвет: темно-синий (#4A7BA7)

Цвета можно изменить в файле `styles.css` в секции `:root`.

## Запуск

1. Откройте `index.html` в браузере
2. Или используйте локальный сервер:

   ```bash
   # Python
   python -m http.server 8000

   # Node.js
   npx http-server
   ```
